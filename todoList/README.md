# Aplicativo Flask Simples

Aplicativo Flask com duas rotas simples configurado para deploy no Vercel.

## Rotas

- `/` - Retorna "olá mundo"
- `/segunda` - Retorna "você está na segunda pagina"

## Deploy no Vercel

1. Instale a CLI do Vercel: `npm i -g vercel`
2. Execute: `vercel`
3. Siga as instruções na tela

Ou faça o deploy diretamente através do site do Vercel conectando seu repositório.
